## 1. Clasificación de temperaturas (-30 a 50)
Este código genera una lista de números entre -30 y 50 y recorre cada valor para clasificarlo según su “temperatura”: números negativos como “Muy frío”, entre 0 y 24 como “Templado”, y mayores a 24 como “Caluroso”.  
Este código utiliza un bucle `for` con `range` y estructuras condicionales (`if`, `elif`, `else`) para recorrer una secuencia de números y clasificarlos según su valor. Se usa este tipo de código porque permite automatizar la evaluación de muchos datos de forma eficiente y asignar categorías según condiciones.

## 2. Números del 1 al 100 con “Bip”
Este programa recorre los números del 1 al 100 e imprime “Bip” si el número es múltiplo de 5 y “Bip Bip” si además es múltiplo de 10, usando el operador módulo (`%`) para identificar divisibilidad.  
Este programa emplea un bucle `for` junto con condiciones y el operador módulo (`%`) para identificar múltiplos de 5 y 10. Este enfoque es adecuado porque permite verificar patrones numéricos (divisibilidad) dentro de una secuencia y ejecutar acciones específicas según el resultado.

## 3. Suma y nivel del usuario
El código solicita dos números al usuario, los suma y clasifica el resultado en tres niveles: “principiante” si es menor a 50, “intermedio” si es menor a 80, y “pro” si es 80 o más.  
El código utiliza entrada de datos (`input`), operaciones aritméticas y condicionales (`if`, `elif`, `else`) para procesar información ingresada por el usuario. Se usa este tipo de estructura porque permite interactuar con el usuario y tomar decisiones en función de los resultados obtenidos.

## 4. Perfil según gustos y edad
Este programa pide la edad y preferencias del usuario (música y baile), convierte las respuestas en valores booleanos y determina un perfil: “estrella de la fiesta” si cumple todas las condiciones, una preferencia musical sin baile, o un estilo propio en otros casos.  
Este programa combina entrada de datos, conversión a valores booleanos (de respuesta si/no) y condicionales lógicas (`and`, `not`) para evaluar múltiples criterios al mismo tiempo. Se utiliza este tipo de código porque permite analizar preferencias del usuario y generar resultados personalizados según varias condiciones simultáneas.
